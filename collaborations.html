  • regioneR, an R package for the management and comparison of genomic regions. It offers a set of function for basic manipulation of region sets extending the functionality of GenomicRanges and a powerful and customizable permutation test framework. With it, it's possible to study the association of a set of regions with other sets of regions, functions defined over the genome or essentially any user defined function
